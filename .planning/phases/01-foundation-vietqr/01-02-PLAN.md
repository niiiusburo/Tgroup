```sql
  id SERIAL PRIMARY KEY,
  bank_bin VARCHAR(20) NOT NULL,
  bank_number VARCHAR(50) NOT NULL,
  bank_account_name VARCHAR(200) NOT NULL,
  updated_at TIMESTAMP DEFAULT NOW()
);
```

Create `api/src/routes/bankSettings.js`:
- `GET /api/settings/bank`: Query `SELECT bank_bin, bank_number, bank_account_name FROM company_bank_settings ORDER BY id LIMIT 1`. Return JSON `{ bankBin, bankNumber, bankAccountName }` or 404 if no row.
- `PUT /api/settings/bank`: Expect body `{ bankBin, bankNumber, bankAccountName }`. Validate all three fields are non-empty strings; if not, return 400. Query `SELECT id FROM company_bank_settings ORDER BY id LIMIT 1`. If row exists, run `UPDATE company_bank_settings SET bank_bin = $1, bank_number = $2, bank_account_name = $3, updated_at = NOW() WHERE id = $4`. If no row, run `INSERT INTO company_bank_settings (bank_bin, bank_number, bank_account_name) VALUES ($1, $2, $3)`. Return JSON `{ success: true, bankBin, bankNumber, bankAccountName }`.
